HELLO VUE !

Vue is framework which lets you create static and dynamic web pages. it hels you to build SPA and helps you to save lots of time in your development process.

Vue is a javascript framework so you need to have a little bit understanding of JS for getting into vuejs.

You don't need to be an javascript expert to try vue.

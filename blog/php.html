What is a Variable ?

A variable is just like an empty box in which we can put anything, so that in future if we require that we can just use that box or (variable).

Imagine yourself as a variable you have a name, age, city, school and so on.

In PHP we declare variables by $ sign $nameofvariable.

Type of variables / Data types

In any programming language, we have different types of data to deal with sometimes we have a string, sometimes we have numbers, sometimes ve have boolean values (True or False) so to store these we have different types of data types in PHP.
